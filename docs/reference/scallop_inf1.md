# Supplementary table 3

Supplementary information for Zhao et al. (2023) .

## Usage

``` r
scallop_inf1
```

## Format

A data frame with 180 rows and 19 variables:

-   UniProt:

    UnitProt ID

-   Protein:

    Protein name

-   Protein_gene_symbol:

    Gene symbol

-   Chromosome:

    Chromosome

-   Position:

    Position

-   cistrans:

    cis/trans

-   rsid:

    reference sequence ID

-   Effect_allele:

    Effect allele

-   Other_allele:

    Eeference allele

-   EAF:

    Effect allele frequency

-   b:

    b

-   SE:

    SE

-   log10P:

    log10(P)

-   Direction:

    Direction field in METAL output

-   HetISq:

    *I*\\(^2\\)

-   HetChiSq:

    Heterogeneity chi-square

-   HetDf:

    degrees of freedom

-   logHetP:

    Heterogeneity log10(P)

-   N:

    N
