# SomaScan 11k

This is also the latest panel

## Usage

``` r
SomaScan11k
```

## Format

A data frame with 10,776 rows and 5 variables:

-   `Sequence.ID`:

    Sequence ID

-   `Full.Name`:

    Full name

-   `Target.Name`:

    Target name

-   `UniProt.ID`:

    UniProt ID

-   `Entrez.Gene.Name`:

    Entrez gene name
