# Supplementary table 4

Supplementary information for Sun et al. (2018) Nature

## Usage

``` r
st4
```

## Format

A data frame with 1980 rows and 31 variables:

-   `Locus.ID`:

    Locus id

-   `SOMAmer.ID`:

    SOMAmer id

-   `Target`:

    Protein

-   `Target.fullname`:

    Protein full name

-   `UniProt`:

    UniProt id

-   `Sentinel.variant*`:

    Sentinel variant

-   `Chr`:

    Chromosome

-   `Pos`:

    Position

-   `Region.start`:

    Region start

-   `Region.end`:

    Region end

-   `Effect.Allele.(EA)`:

    Effect allele

-   `Other.Allele.(OA)`:

    Other allele

-   `EAF`:

    Effect allele frequency

-   `INFO`:

    Information score

-   `cis/.trans`:

    "cis"/"trans"

-   `Mapped.gene`:

    Mapped gene

-   `No..conditionally.significant.variants`:

    Number of conditionally significant variants

-   `Previously.reported`:

    Previously reported

-   `Replicates?`:

    Yes/No

-   `beta`:

    b

-   `SE`:

    SE

-   `p`:

    p value
