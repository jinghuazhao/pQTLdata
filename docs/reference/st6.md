``` r
st6
```

## Format

A data frame with 163 rows and 20 variables:

-   `Locus.ID`:

    Locus id

-   `Sentinel.variant*`:

    Sentinel variant

-   `Chr`:

    Chromosome (1-12,14-20,22)

-   `Pos`:

    Position

-   `SOMAmer.ID`:

    SOMAmer id

-   `Target`:

    Protein

-   `Target.fullname`:

    Protein full name

-   `UniProt`:

    UniProt id

-   `cis/.trans`:

    "cis"/"trans"

-   `Mapped.gene`:

    Mapped gene

-   `Effect.Allele.(EA)`:

    Effect allele

-   `Other.Allele.(OA)`:

    Other allele

-   `Previously.reported`:

    Previously reported (0/1)

-   `beta`:

    b

-   `SE`:

    SE

-   `p`:

    p value
