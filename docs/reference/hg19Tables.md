# hg19 Table

Gene information

## Usage

``` r
hg19Tables
```

## Format

A data frame with 19872 rows and 12 variables:

-   `X.chrom`:

    Chromosome

-   `chromStart`:

    chromStart

-   `chromEnd`:

    chromEnd

-   `strand`:

    Strand

-   `acc`:

    UniProt id

-   `uniprotName`:

    Protein

-   `accList`:

    List of UniProt ids

-   `isoIds`:

    isoIds

-   `geneName`:

    geneName

-   `geneSynonyms`:

    geneSynonyms

-   `hgncSym`:

    HGNC symbol

-   `ensGene`:

    ENSEMBL gene

## Details

Curation from UCSC.
