# hg19 information

protein information

## Usage

``` r
hg19
```

## Format

A data frame with 62559 rows and 8 variables:

-   `chr`:

    Chromosome (chr1-22,X,Y,...)

-   `start`:

    start

-   `end`:

    end

-   `width`:

    width

-   `strand`:

    strand

-   `ENSEMBL`:

    ENSMEBL gene

-   `SYMBOL`:

    HGNC symbol

-   `UNIPROT`:

    UniProt id
