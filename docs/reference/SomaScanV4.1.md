# SomaScan v4.1

This is the 7k panel

## Usage

``` r
SomaScanV4.1
```

## Format

A data frame with 7,288 rows and 6 variables:

-   `#`:

    A serial number

-   `SeqID`:

    SeqID

-   `Human.Target.or.Analyte`:

    Human target/analyte

-   `UniProt.ID`:

    UniProt id

-   `GeneID`:

    HGNC symbol

-   `Type`:

    "Protein"

## Details

obtained directly from SomaLogic.
