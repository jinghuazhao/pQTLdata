# Seer 1980 panel

ST1 from Suhre et al. (2024) .

## Usage

``` r
seer1980
```

## Format

A data frame with 1,980 rows:

-   `PID.NP`:

    PID.NP

-   `protein_ids`:

    protein_ids

-   `protein_names`:

    protein_names

-   `mapped.UniProtID`:

    mapped.UniProtID

-   `mapped_gene_id`:

    mapped_gene_id

-   `gene_name`:

    gene_name

-   `description`:

    description

-   `chr`:

    chr

-   `start`:

    start

-   `end`:

    end

## Details

As above.
