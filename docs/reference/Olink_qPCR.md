# Olink/qPCR panels

Information on all qPCR panels

## Usage

``` r
Olink_qPCR
```

## Format

A data frame with 1,112 rows and 7 variables:

-   `UniProt`:

    UniProt id

-   `Panel`:

    Panels

-   `Target`:

    Protein

-   `gene`:

    HGNC symbol

-   `chr`:

    Chromosome

-   `start`:

    start

-   `end`:

    end
