# Olink/Target 96 panels

Information on all Target 96 panels. Individual panels are also
available from the companion xlsx in the Olink/ directory.

## Usage

``` r
Olink_Target_96
```

## Format

A data frame with 1,116 rows and 3 variables:

-   `UniProt`:

    UniProt id

-   `Protein`:

    Protein

-   `Panel`:

    Panel
