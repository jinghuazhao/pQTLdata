# Olink/Explore HT panels

Information on all qPCR panels

## Usage

``` r
Olink_Explore_HT
```

## Format

A data frame with 5,416 rows and 4 variables:

-   `Olink.ID`:

    Olink id

-   `UniProt.ID`:

    UniProt id

-   `Protein.name`:

    Protein name

-   `Gene.name`:

    Gene name
